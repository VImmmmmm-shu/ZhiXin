制作合体的衣服并实现换装功能（4/3）
通过复制身体视觉网格作为独立的“衣服组”，然后在换装时显示对应的衣服组，同时隐藏原始身体部件，实现合体换装。
具体操作步骤
第一步：收集原始身体的所有视觉网格
1.	在 Hierarchy 中，找到 G1 机器人（最外层对象）。
2.	在 Hierarchy 的搜索框中输入 t:MeshRenderer，筛选出所有带有 MeshRenderer 组件的子物体。
3.	逐个选中这些物体，在 Scene 视图中观察它们的位置，判断哪些是身体外观部分（如躯干、四肢、头部），哪些是碰撞体、辅助物体（名称可能含 Collisions、Physics 等）。
o	通常，真正的视觉网格名称会包含 Visuals、Mesh、Renderer，或者位于 Visuals 文件夹下。
o	例如：pelvis_0、pelvis_contour_link_0 1、torso_link 下的 Visuals 2 等。
4.	按住 Ctrl 键多选所有属于身体外观的物体（排除碰撞体和辅助物体）。
o	如果不好区分，可以先全选，然后在 Scene 视图中观察，那些看起来像是身体部分的就是正确的。
o	确保不要选中任何名字带 Collisions 的物体，它们只是物理碰撞体，不是视觉网格。
5.	将这些选中的物体拖入到一个新建的空物体中，以便后续统一管理：
o	在 Hierarchy 中右键 → Create Empty，命名为 DefaultBody。
o	将选中的所有视觉网格物体拖入 DefaultBody 下（保持它们的相对位置不变）。
o	此时，DefaultBody 包含了机器人身体的全部外观部件。
第二步：复制身体外观作为衣服组
1.	在 Hierarchy 中选中 DefaultBody，按 Ctrl+D 复制一次，得到 DefaultBody (1)。
2.	将副本重命名为 RedSuit。
3.	再次复制 DefaultBody，得到第二个副本，命名为 BlueSuit。
4.	再次复制 DefaultBody，得到第三个副本，命名为 RobotArmor。
5.	现在，您有三个衣服组，每个组都包含一份完整的身体视觉网格副本。
第三步：为每个衣服组创建独立材质
1.	在 Project 窗口中创建三个材质：（在Assets/Meterial 目录）
o	右键 → Create → Material，分别命名为 RedCloth、BlueCloth、ArmorMaterial。
o	设置材质颜色：红色、蓝色、灰色（或金属灰）。
2.	为 RedSuit 组的所有物体更换材质：
o	在 Hierarchy 中展开 RedSuit，选中其下的所有子物体（可以通过搜索 t:MeshRenderer 快速多选）。
o	在 Inspector 中，将它们的 MeshRenderer 组件的 Materials 列表中的材质替换为 RedCloth。
o	可以一次性多选后，在 Inspector 中修改，或者写一个简单脚本批量替换（推荐手动逐一检查，确保所有部件都换色）。
3.	同样的操作为 BlueSuit 和 RobotArmor 更换对应的材质。
第四步：修改 Wardrobe 脚本以控制衣服组
更新 Wardrobe.cs，让它控制三个衣服组的显隐，并隐藏原始身体组。
csharp
using UnityEngine;

public class Wardrobe : MonoBehaviour
{
    public GameObject defaultBody;      // 原始身体组（DefaultBody）
    public GameObject[] clothGroups;    // 衣服组（RedSuit, BlueSuit, RobotArmor）
    private int currentIndex = 0;

    void Start()
    {
        SwitchTo(0);
    }

    public void SwitchTo(int index)
    {
        if (index < 0 || index >= clothGroups.Length) return;

        // 先隐藏所有衣服组
        foreach (var group in clothGroups)
            if (group != null) group.SetActive(false);
        // 显示原始身体组
        if (defaultBody != null) defaultBody.SetActive(true);

        // 激活指定的衣服组，并隐藏原始身体组
        if (index >= 0 && index < clothGroups.Length)
        {
            clothGroups[index].SetActive(true);
            if (defaultBody != null) defaultBody.SetActive(false);
        }

        currentIndex = index;
        Debug.Log("切换到衣服索引: " + index);
    }

    public void SwitchTo(string outfitName)
    {
        // 根据名称映射索引（简单示例，可根据需要扩展）
        if (outfitName.Contains("红") || outfitName.Contains("red")) SwitchTo(0);
        else if (outfitName.Contains("蓝") || outfitName.Contains("blue")) SwitchTo(1);
        else if (outfitName.Contains("灰") || outfitName.Contains("灰") || outfitName.Contains("铠甲") || outfitName.Contains("金属")) SwitchTo(2);
        else Debug.LogWarning("未匹配到颜色关键词");
    }

    public void NextCloth()
    {
        int next = (currentIndex + 1) % clothGroups.Length;
        SwitchTo(next);
    }
}
第五步：在 Inspector 中配置脚本
1.	在 Hierarchy 中选中 G1 对象。
2.	找到 Wardrobe 组件（如果没有，点击 Add Component 添加）。
3.	将 DefaultBody 对象（您第一步创建的空物体）拖入 Default Body 字段。
4.	将 Cloth Groups 数组大小设为 3，依次拖入：
o	Element 0：RedSuit
o	Element 1：BlueSuit
o	Element 2：RobotArmor
